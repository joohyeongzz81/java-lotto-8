# java-lotto-precourse

# 기능 목록

## 입력 기능
- 로또 구입 금액 입력받기
- 당첨 번호 입력받기 (쉼표로 구분)
- 보너스 번호 입력받기

## 검증 기능
- 구입 금액이 1,000원 단위인지 검증
- 구입 금액이 양수인지 검증
- 로또 번호가 1~45 범위인지 검증
- 로또 번호가 6개인지 검증
- 로또 번호에 중복이 없는지 검증
- 보너스 번호가 1~45 범위인지 검증
- 보너스 번호가 당첨 번호와 중복되지 않는지 검증

## 로또 발행 기능
- 구입 금액으로 로또 장수 계산
- 1~45 사이 중복되지 않는 6개 숫자 생성
- 로또 번호 오름차순 정렬

## 당첨 확인 기능
- 로또 번호와 당첨 번호 일치 개수 확인
- 보너스 번호 일치 여부 확인
- 당첨 등수 판정 (1~5등)

## 통계 기능
- 등수별 당첨 개수 집계
- 총 상금 계산
- 수익률 계산 (소수점 둘째 자리 반올림)

## 출력 기능
- 구매한 로또 개수 출력
- 로또 번호 출력 (오름차순)
- 당첨 통계 출력
- 수익률 출력

## 예외 처리
- 잘못된 입력 시 "[ERROR]" 메시지 출력
- 예외 발생 시 해당 부분부터 재입력
